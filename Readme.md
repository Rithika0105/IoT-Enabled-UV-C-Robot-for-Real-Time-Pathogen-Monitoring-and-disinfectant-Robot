# IoT Enabled UV-C Disinfection Robot

## 📌 Description
This project is an IoT-based robot used for disinfection using UV-C light and real-time monitoring.

## 🚀 Features
- WiFi control using ESP32
- Motor movement control
- UV-C disinfection
- Sensor monitoring

## 🔧 Components
- ESP32
- L298N Motor Driver
- DC Motors
- Relay Module
- Sensors
